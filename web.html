What is Web-Development?

Web-Development, also known as website development, encompasses the processes involved in crafting, constructing and sustaining websites and web applications that run online on a browser. This field may encompass tasks such as web design, web programming and managing databases. Fundamental tools utilized in web development include programming languages like HTML (Hypertext Markup Language), CSS (Cascading Style Sheets) and JavaScript.

Hypertext Markup Language (HTML)

This is a text-based markup language for structuring digital documents. HTML documents can be loaded from the World Wide Web (WWW) and displayed with a web browser. In addition to the actual content, they can also contain meta-information (language, author, keywords, etc.). So-called tags describe document elements (text, images, etc.).

Cascading Style Sheets (CSS)

CSS is a stylesheet language used for describing the presentation of a document written in a markup language such as HTML. CSS can be embedded directly into HTML documents or written into external text files. With CSS, various style attributes can be defined for HTML elements. Style statements are inherited, i.e. elements adopt the definitions of their parent element. These statements are specified in property-value pairs.

CSS important features:

  • Font: font-family, font-size, font-color
  • Element width: width
  • Element height: height
  • Distance (outside): margin
  • Distance (inside): padding
  • Border: border
  • Transparency: opacity
  • Line-height: line-height
  • Comment: /* Comment */

CSS example:

The css example from W3schools css:

<h1 style="color:blue;">A Blue Heading</h1>
<p style="color:red;">A red paragraph.</p>

JavaScript

JavaScript is a versatile scripting language that enables interactive web pages and is an essential part of web applications. Unlike HTML, which structures content and CSS, which handles page style, JavaScript provides web pages with dynamic behavior. This can include anything from animated graphics, interactive forms, and live content updates without needing to reload the page.

JavaScript runs in the client's browser and can interact with the document content through the Document Object Model (DOM), allowing developers to create rich, interactive web experiences. It can also communicate with servers, enabling web applications to be more responsive and dynamic by loading new data in the background.

Key Concepts of JavaScript:

  • Variables and data types
  • Operators and expressions
  • Functions and scope
  • DOM manipulation
  • Events and event handling
  • Asynchronous programming and Promises
  • APIs and AJAX

Overall, JavaScript's ability to modify both HTML and CSS and its control over the browser, makes it a crucial tool for creating dynamic and interactive web applications.
