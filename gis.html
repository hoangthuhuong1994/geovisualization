What is GIS?

As a defintion of ESRI , GIS is a system that creates, manages, analyzes, and maps all types of data. GIS connects data to a map, integrating location data (where things are) with all types of descriptive information (what things are like there). This provides a foundation for mapping and analysis that is used in science and almost every industry. GIS helps users understand patterns, relationships, and geographic context. The benefits include improved communication and efficiency as well as better management and decision making. There are main two GIS will be describe in this porfolio which are QGIS and ArcGIS Pro

Quantum Geographic Information System (QGIS)

QGIS is a open-source software for creating, editing, visualizing, analyzing and publishing geospatial information. It supports a wide range of formats and offers tools for both vector and raster data, making it versatile for geography, urban planning and environmental science.

QGIS is available on Windows, macOS and Linux, it supports a wide range of data formats and offers extensive customization options through plugins. The QGIS documentation can be found in here: Documentation for QGIS 3.34

ArcGIS Pro

Developed by Esri, it is a leading tool in geospatial analysis, offering advanced capabilities for mapping, spatial analysis, and data management. ArcGIS Pro allows users to store multiple items, such as maps, layouts, tables, and charts, in a single project and work with them as needed.

The ArcGIS Pro documentation can be found in here: Introduction to ArcGIS Pro

QGIS and ArcGIS Pro Comparison

Table 1: The comparison between QGIS and ArcGIS Pro
Feature QGIS ArcGIS Pro
Cost Free and Open-Source Subscription-based
Platform Compatibility Windows, macOS, Linux Windows
User Interface Customizable, Community-driven Plugins Modern, Ribbon-based, Integrated with ArcGIS platform
Data Format Support Extensive with native support Extensive, with additional support through ArcGIS platform
Analysis and Visualization Tools Comprehensive, Extensible through plugins Advanced, out-of-the-box functionality
Community and Support Large, active community; relies on community support Professional support from Esri; extensive documentation and training resources

Exercise 2: Map of the North-Pole region

The main task of the exercise involves creating a map of the North Pole region to visualize the drift data of the R/V POLARBJORN ship from the Coordinated Eastern Arctic Experiment (CEAREX) during 1988-1989. Which involves:

  • Data Preparation: Importing CEAREX ASCII files, including the PBDRIFT.POS file into QGIS. These files contain detailed positions of ship and buoy drifts, crucial for analysis and visualization.
  • Data Import and Processing: Loading the drift data and additional vector files like the world's coastlines and a graticule dataset for enhanced visualization.
  • Projection Selection: North_Pole_Orthographic with id "ESRI:102035" to reduce distortion in the polar region and improve visual accuracy.
  • Visualization in QGIS
  • Extending Visualization with Line Features
  • Time Attribute Processing and Animation
